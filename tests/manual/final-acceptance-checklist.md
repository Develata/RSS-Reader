## US1 核心阅读路径

- [ ] 可以添加有效 feed URL。
- [ ] 可以刷新订阅且不会重复写入已有文章。
- [ ] 文章列表能展示刷新结果并默认按发布时间排序。
- [ ] 阅读页优先展示完整 HTML 正文。
- [ ] 本地缓存文章在离线情况下仍可打开。
- [ ] 失败场景会显示清晰错误提示，不导致应用崩溃。

## US2 阅读进度管理

- [ ] 文章可标记已读/未读，并在重启后保持。
- [ ] 文章可收藏/取消收藏，并在重启后保持。
- [ ] “仅未读”“仅收藏”和标题搜索可组合使用。
- [ ] 阅读页快捷键 `M`、`F` 与按钮行为一致。

## US3 配置交换

- [ ] 配置包 JSON 可成功导出与导入。
- [ ] OPML 可成功导出与导入。
- [ ] WebDAV 上传与下载可完成配置交换。
- [ ] 导入或同步后仅恢复订阅与设置，不恢复文章库和状态。
- [ ] 损坏、重复或未知字段配置会被拒绝。
