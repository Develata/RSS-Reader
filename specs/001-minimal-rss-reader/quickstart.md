# 快速开始：极简个人 RSS 阅读器 MVP

## 1. 准备环境

- 安装 Rust 稳定版工具链
- 准备支持桌面/Web/Android 的 Dioxus 开发环境
- 确保本地可用 SQLite

## 2. 初始化项目

1. 在仓库根目录创建 Rust workspace。
2. 创建以下 crate：
   - `crates/rssr-app`
   - `crates/rssr-application`
   - `crates/rssr-domain`
   - `crates/rssr-infra`
3. 配置共享依赖、lint 和格式化工具。

## 3. 实现基础能力

1. 创建 SQLite schema 与迁移。
2. 定义订阅源、文章、用户偏好设置和配置包模型。
3. 实现订阅抓取、RSS/Atom 解析和本地仓储。
4. 实现配置导入导出与 OPML 支持。

## 4. 验证 MVP 核心路径

1. 添加一个有效 feed URL。
2. 刷新订阅并写入本地文章。
3. 打开文章列表并进入阅读页。
4. 切换已读/收藏状态并重启应用验证持久化。
5. 按标题搜索文章。
6. 导出配置包与 OPML，再重新导入验证。

## 5. 验证配置交换

1. 配置远端位置。
2. 上传当前配置包。
3. 清空本地订阅和设置或使用新环境。
4. 下载并导入配置包。
5. 确认订阅源和偏好设置恢复，但文章库和状态不被同步。

## 6. 手工体验检查

- 桌面端快捷键是否可用
- 列表滚动是否顺滑
- 阅读页排版是否清晰
- 浅色/深色主题切换是否正常
- Android 触控导航是否自然
